Webhook Queue Server

This server stores, logs, and replays webhook payloads for registered projects.

  • POST /register - Register project
  • POST /set-target/{project_id} - Set where to forward hooks
  • POST /hook/{project_id} - Accepts payloads
  • GET /logs/{project_id} - View payload logs
  • POST /replay/{project_id} - Replay all logs to target
